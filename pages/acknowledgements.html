Data source & attribution

This project is built entirely on public, de-identified research data and open-source software. The sources below made the analysis possible and are credited as their data-use terms request.

Data

The results shown here are in whole or part based upon data generated by the TCGA Research Network: https://www.cancer.gov/tcga.

The whole-slide images were obtained from the NCI Genomic Data Commons (GDC). All images and clinical/omics data are de-identified and publicly available for research use.

Grossman RL, Heath AP, Ferretti V, Varmus HE, Lowy DR, Kibbe WA, Staudt LM. Toward a Shared Vision for Cancer Genomic Data. N Engl J Med. 2016;375(12):1109–1112. doi:10.1056/NEJMp1607591

Software & methods

Patch-level features were extracted with the UNI2 pathology foundation model (1,536-dim embeddings per patch). The analysis and this website were built with open-source tools:

  • NumPy, pandas, SciPy — numerical computing and data handling.
  • scikit-learn — PCA, classifiers, cross-validation.
  • scikit-survival & lifelines — survival models, the concordance index and Kaplan–Meier / log-rank analysis.
  • h5py — reading the patch-embedding archives.
  • Matplotlib & seaborn — the figures.
  • Chart.js — the interactive charts on this site.

Intended use

This is an educational mentorship project. Everything here is for research and educational purposes only and is not intended for clinical or diagnostic use.
