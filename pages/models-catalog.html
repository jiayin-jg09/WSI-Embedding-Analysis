The association catalog

The Biology page shows that UNI2 embeddings encode molecular state. This page is the detailed registry: the single strongest dimension–molecule links, each given a stable code, scored for how well they reproduce across six independent foundation models, and checked against patient survival. It is meant to be browsed and sorted, not read top to bottom.

How to read an entry

Every row is one association between a single UNI2 embedding dimension and a single molecular feature. Each is given a stable code so it can be referred to later:

U2 · D0953 · EXPR · 7A3F2
model (UNI2)  ·  dimension 953  ·  data type (expression)  ·  association ID

The columns:

  • rho (within-cancer) — the Spearman correlation between the dimension and the feature after cancer type is removed. A positive value means the dimension rises with the feature; the sign and magnitude both matter.
  • models — the cross-model robustness count, from 0 to 6 (see below). Higher means more foundation models independently rank this feature among their best-encoded, so the link is unlikely to be a quirk of UNI2.
  • surv HR/SD — the hazard ratio per standard deviation of that dimension, from a Cox model stratified by cancer type. Values above 1 mean higher values of the dimension go with worse survival; * marks p < 0.05.

How the catalog is built

Within-cancer strength. Pooling all cancers together inflates correlations, because a dimension that merely separates colon from liver tumors will line up with anything else that differs between them. So every association is measured within cancer type first, and only the genuinely strongest survive into the catalog.

Balanced across data types. Immune-signature correlations are numerically much larger than, say, copy-number ones, so a simple global top-1000 would be almost entirely immune signatures. Instead the catalog keeps the strongest ~200 links from each data type (expression, copy-number, immune fractions, immune signatures, proteins), so every modality is represented.

Cross-model robustness. The mentor's lab computed the same dimension– omics correlations for six pathology foundation models (CTransPath, Kaiko-S8, Lunit-DINO, PathDINO, SP22M and UNI2). Their embedding dimensions are not comparable one to one, so we compare at the feature level: for each model we ask whether the feature is among that model's top 20% best-encoded, and count how many of the six agree. A link confirmed by all six is reading biology that is a property of the tissue rather than of any single network.

Survival. For each dimension in the catalog we fit a Cox proportional-hazards model on overall survival, stratified by cancer type, and record the hazard ratio and significance. This flags which appearance-encoded signals also track patient outcome.

Hub dimensions. The links are also grouped by dimension to surface the “hub” dimensions that carry many strong associations at once. Mutations are catalogued separately, because within cancer type the embedding barely tracks specific mutations.

What is and isn't new here

To be clear about novelty: the raw dimension–omics correlations are not a new discovery. The lab already holds them, pooled, for all six models including UNI2. What this catalog contributes is the curation on top: the de-confounded within-cancer view, the cross-model agreement score, the survival link, the program labels, and stable codes to refer to individual associations. These are also correlations, not proof that the model “reads a gene”; a dimension may track a gene because both follow the same tissue morphology.
